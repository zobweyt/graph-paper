# Идеальная тетрадная клетка (A4)

Сделал для себя генератор «умной» разметки, когда стандартные покупные тетради бесят своей яркой синей клеткой или кривыми полями. Это максимально чистый лист для тех, кто любит эстетику и порядок в конспектах.

### В чём фишка

Линии здесь — это не линии в обычном понимании. Это **микроскопические точки**, выстроенные в ряд. 

- **Для глаз:** Клетка кажется очень светлой и не отвлекает от записей. Читать написанное поверх такой сетки — одно удовольствие.
- **Для сканов:** Это «обман» для алгоритмов обработки изображений. При сканировании (например, через телефон) точки становятся невидимыми. На выходе получается идеально чистый белый лист с вашим текстом, как будто сетки и не было.
- **Для принтера:** Принтеру проще выдать чёткую точку, чем пытаться «пылить» бледную серую линию. Это даёт максимальную чистоту печати.

### Как пользоваться

1. Скачайте файл `index.html`.
2. Откройте его в любом браузере.
3. Нажмите `Ctrl + P` (Печать).
4. Выберите **«Сохранить как PDF»**.
5. Полученный PDF отправляйте на печать.

### Тонкая настройка

Принтеры у всех разные, поэтому параметры стоит один раз подобрать под себя. Откройте файл через блокнот или VS Code и поправьте эти значения в скрипте:

* `pointsPerSide` — сколько точек будет в одной стороне клетки (10 — оптимально).
* `radius` — размер самой точки. 

По умолчанию стоит `0.06`. Если на вашем принтере клетку почти не видно — увеличьте до `0.08` или `0.1`. Если она слишком жирная — уменьшите.
